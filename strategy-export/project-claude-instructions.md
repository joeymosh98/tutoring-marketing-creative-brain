# Tutero Online Tutoring LP

## Security

- **NEVER commit secrets or API keys.** All sensitive values belong in `.env` (gitignored). Reference `.env.example` for the expected variable names.
- Do not hardcode API keys, tokens, or passwords anywhere in the codebase.
- If you need a new secret, add the variable name (without the value) to `.env.example` and instruct the user to set it in `.env`.

## Ad Copy Rules

- **NEVER call tutors "teachers".** Tutero tutors are not qualified teachers. Use "tutor" in all ad copy, ticks, headlines, and body text. "Matched with a real tutor" — not "matched with a real teacher."
