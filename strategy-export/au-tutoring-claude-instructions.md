# Tutero Tutoring Australia — Skills

Parent-facing 1-on-1 online tutoring for the Australian market (Tutero.com).

## Skills

1. **tutero-tutoring-parent-copy-australia.md** — Copywriting rules: master narrative, five pillars, forbidden words, thinking moves, and pre-flight checklist for all parent-facing copy.

2. **paid-social-winner-insights-australia.md** — Ad creation rules: pattern library from 12 proven winning Meta ads, seven recurring moves, format heuristics, and scoring checklist.

3. **tutoring-image-generator-australia.md** — Image generation rules for ad and landing page visuals. Reference files in `tutoring-image-generator-references/` (aesthetic rules, AU winning compositions, persona templates, regions & platforms).

4. **creating-paid-social-ads-tutoring-australia.md** — End-to-end ad creation pipeline: cognition system, design system (templates, gradients, spacing), copy rules, image generation, HTML build, anti-patterns, production checklist, and workflow.

## Agents

Agents orchestrate skills to produce deliverables. They define a persona, workflow, and quality gates.

- **agents/paid-social-ads.md** — **Zara**, the paid social creative producer. Uses all 5 skills (plus `tutero-brand`) to produce Meta/Facebook/Instagram static ad creatives end-to-end. Activate when asked to create ads.

## Usage

- Read **paid-social-winner-insights-australia.md** FIRST when creating ads (what patterns actually convert)
- Read **tutero-tutoring-parent-copy-australia.md** for voice, forbidden words, and narrative
- Read **tutoring-image-generator-australia.md** when generating images for ads or landing pages
- Read **creating-paid-social-ads-tutoring-australia.md** for the full build pipeline
- All skills should be consulted together for any parent-facing creative
- When creating ads, activate the **paid-social-ads agent** — it orchestrates the correct reading order and workflow
