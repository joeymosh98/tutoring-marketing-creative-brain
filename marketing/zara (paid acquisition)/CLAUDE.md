# Zara — Paid Acquisition

Parent-facing 1-on-1 online tutoring for the Australian market (Tutero.com).

## Skills

1. **parent-copy.md** — Copywriting rules: master narrative, five pillars, forbidden words, thinking moves, and pre-flight checklist for all parent-facing copy.

2. **winner-insights.md** — Ad creation rules: pattern library from 12 proven winning Meta ads, seven recurring moves, format heuristics, and scoring checklist.

3. **image-generator.md** — Image generation rules for ad and landing page visuals. Reference files in `references/` (aesthetic rules, AU winning compositions, persona templates, regions & platforms).

4. **creating-paid-social-ads.md** — Orchestrator for the full ad creation pipeline: cognition system (6-step thinking process) and production workflow (9-step build sequence). References these companion files:
   - **design-system.md** — Brand constants, 6 templates (A–F) with full CSS, file structure, ads.json schema, gallery page.
   - **copy-and-image-rules.md** — Eyebrow/tick copy rules, tick pools by pillar, image generation API setup, anti-AI prompts, photo composition.
   - **quality-gates.md** — 16 anti-patterns learned from iteration, per-ad checklist, batch checklist.

5. **style-inspiration.md** — 12 layout blueprints reverse-engineered from high-performing ads outside tutoring. Use for experimental formats beyond the core templates.

6. **video-ads-voiceover.md** — End-to-end production skill for voiceover-style video ads: Seedance 2.0 clips + ElevenLabs AU voice (Emma) + lo-fi acoustic music bed + ffmpeg assembly. Covers cognition system (reason to buy, persona, arc, VO script, shot list), production workflow with cost controls and rework protocol, and quality gates. References `creating-paid-social-ads` for shared cognition (reasons to buy, personas), `parent-copy` for voice rules, and `visual-identity` for end card design.

## Dependencies

- **visual-identity** (Ella) — brand colours, typography, logos, tone of voice, the Tutoring Ad Aesthetic recipe
- **company strategy** — group strategy, Australia market context

## Reading Order

1. Read **winner-insights.md** FIRST when creating ads (what patterns actually convert)
2. Read **parent-copy.md** for voice, forbidden words, and narrative
3. Read **image-generator.md** when generating images for ads or landing pages
4. Read **creating-paid-social-ads.md** for the full build pipeline, then its companions: **design-system.md**, **copy-and-image-rules.md**, **quality-gates.md**
5. Consult **style-inspiration.md** for experimental layouts
6. Read **video-ads-voiceover.md** when producing video ads (voiceover-style only — not UGC or talking-head)
7. All skills should be consulted together for any parent-facing creative
